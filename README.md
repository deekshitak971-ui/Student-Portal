DESCRIPTION :
The student portal is a frontend web application designed to help students easily access academic information and service in one place. The portal provides features such as viewing 
student profiles,checking course details,accessing announcements,and managing basic academic information
The main goal of the portal is to make it easier for student to interact with academic resources
online.
